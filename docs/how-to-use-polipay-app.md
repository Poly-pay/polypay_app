# How to use Polypay App

## Polypay App (Beta v0.1)

Polypay app beta v0.1 is an initial release of the Polypay platform. It is a web application that allows you to create and manage a multisig wallet on **Ethereum Sepolia testnet** with a focus on privacy. It is not yet production ready. Current version includes the following features:

- Create and manage a multisig wallet
- Hide signers(multisig wallet owners) identities with ZK proofs
- Execute payroll payments: Transfer funds to multiple recipients
- View transaction history
- View multisig wallet balance
- View multisig wallet address

### User Workflow

1. **Connect Wallet**: Connect your Ethereum wallet
2. **Generate Identity**: Sign a message to create your secret
3. **Create/Join Wallet**: Deploy new multisig or join existing one
4. **Propose Transaction**: Create transfer and generate ZK proof
5. **Sign**: Other signers approve with their ZK proofs
6. **Execute**: When threshold reached, execute the transaction
